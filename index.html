Medical Image Analysis in the Cloud

Get Started

An open source project

to deploy, compare and access deep learning models for medical image analysis over the network.
Everything runs in the cloud, so that you can get results everywhere without complications.
TOMAAT provides you with both server-side and client-side packages to make this possible.
